# Cinetech :

Cinetech est une application web permettant de consulter des informations sur des films via une API externe.
Ce document explique étape par étape comment déployer le projet en local, configurer l’API et publier le projet sur GitHub.

L’objectif de ce guide est de permettre à n’importe quel développeur de cloner le projet et de le faire fonctionner sans difficulté.

---

## Objectifs du projet :

- Déployer le projet Cinetech en local
- Configurer correctement l’API via des variables d’environnement
- Publier le projet sur GitHub
- Fournir une documentation claire et compréhensible en Markdown

---

## Prérequis :

Avant de commencer, assurez-vous d’avoir les outils suivants installés :

- **Node.js** (version 18 ou supérieure)
- **npm**
- **Git**
- Un éditeur de code (VS Code recommandé)

Vérification des versions :

```bash
node -v
npm -v
git --version
